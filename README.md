# Power Decompiler
Disassembly and decompilation suite tuned for working with GameCube/Wii binaries. Work in progress!

## Building
Clone and make
```
git clone https://git.shiiion.me/vyuuui/ppc-decomp.git
cd ppc-decomp
make -j
```
Writes the product to `<repository>/bin/decompile`
